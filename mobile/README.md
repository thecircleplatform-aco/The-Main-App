# Mobile (Android / iOS)

Native app wrappers for Circle are provided by **Capacitor**. For Capacitor compatibility, the native projects remain at the **repository root**:

- **Android:** `../android`
- **iOS:** `../ios`

Use the npm scripts from the project root:

- `npm run mobile` — sync and open Android Studio
- `npm run mobile:ios` — sync and open Xcode (Mac only)
- `npm run cap:sync` — sync web build to both platforms

See **docs/CAPACITOR.md** for full setup and build instructions.
